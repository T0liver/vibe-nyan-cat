## Getting Started

### Prerequisites

- Node.js 18+ 
- npm

### Installation

```bash

```

### Development

```bash

```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
